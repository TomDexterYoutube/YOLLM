# v2 Release

Time to celebrate the v2 release.

This update focuses on improving speed, performance, and quality.

## About

This project was started so anyone can create their own LLM.

## Getting Started

1. Add your training data as `.txt` files in the `data/training_data` folder.
2. Edit the configuration to suit your needs. A video guide will be made later.

## Notes

There are a few known limitations:

- Install and run scripts are currently provided as `.sh` files.
- There are no `.bat` files available yet for Windows users.
- The "edit config" option requires `nano` to be installed.
- On Windows, or if you do not want to install `nano`, you can edit the `config.py` file using your preferred text editor.

These will be improved in a future update.
